记录Hexo搭建过程遇到的一些问题

问题一：4000端口被占用

解决方法：
1.可以用hexo server -p 5000 命令将端口修改为5000
2.可以在_config.yml内加上如下代码更改hexo-server运行时的端口号：

server:
port: 4001
compress: true
header: true
其中4001可以换成想要运行的端口号。
参考链接: https://segmentfault.com/q/1010000008546859?_ea=1678427
